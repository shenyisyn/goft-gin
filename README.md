# Goft-Gin
* 在web框架gin的基础上做的脚手架

## 安装
go get -u github.com/shenyisyn/goft-gin@v0.5.4

0.5.4中gin的版本更新到1.9.0，基本功能不变。 后续版本还在开发中
## 文档在此
[https://www.yuque.com/jtthink/xrb1ph](https://www.yuque.com/jtthink/xrb1ph)

## 功能说明
 控制器、简易依赖注入、中间件、表达式、任务组件等。
 
 后续功能正在发布中.
